# Repository Agent Rules

## Scope
- These style-preservation rules apply only to `quakrs-site` (`quakrs.com` website).
- `quakrs-extension` is independent and is not bound by these website style rules.

## Mandatory pre-read for website UI work
- Before generating UI changes in `quakrs-site`, read `/docs/quakrs-site-style.md`.
- Treat `/docs/quakrs-site-style.md` as the source of truth for visual identity continuity.
